The database named 'torii' consists of the following tables

- admin
  This table contains the details of username and password of administrator
  
- booking
  This table contains of all the details the user has entered while booking the desired package

- carddetails
  This table contains all the details of the card such as card no, cvv, expiry etc

- feedback
  This table contails all the details entered by the user for the improvement of the web project

- reglog
  This table contains the details entered by the user at the time of signup
